title: Bulk author addition
--

h2. Configuration data

h3. repo_bulk_author_addition:delimiter

h4. Expected type: String

The character to be used for separating entries in the bulk author addition screen, defaults to a semi-colon

h3. repo_bulk_author_addition:match_using_names

h4. Expected type: Boolean

Config data for whether to match users by their full name or their citations (e.g. John Smith or Smith, J.). Defaults to false (using citation form).
